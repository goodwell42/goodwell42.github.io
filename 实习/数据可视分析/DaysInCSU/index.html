矩阵分解有三个很明显的用途:

  • 降维处理
  • 缺失数据填补（或者说成“稀疏数据填补”）

张量（tensor）

阶 实例
0 纯量（只有方向）
1 向量（大小和方向）
2 矩阵（数据表）
3 3阶张量（数据立方）
n
  1. rank：number of dimensions
  2. shape: number of rows and columns
  3. type: data type of tensor’s elements
